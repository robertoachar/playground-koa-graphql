# Development

- Clone the repo

```bash
$ git clone https://github.com/robertoachar/playground-koa-graphql.git
```

- Install dependencies

```bash
$ npm install
```

- Run scripts

| Action                 | Usage          |
| ---------------------- | -------------- |
| Start development mode | `npm start`    |
| Lint code              | `npm run lint` |
